# How to use this api from python

```python
import requests
data = {'url':'https://www.youtube.com/@PW-Foundation/videos'}
response = requests.post('https://8rkvysfczg.execute-api.us-east-1.amazonaws.com/yt_scrapper_utkarsh',json=data)
response.json()
```
